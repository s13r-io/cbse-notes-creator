## 1. Mathematics

**NCERT Book**: *Mathematics* (NCERT Class 9)
**Content Type**: Formula-based
**Structural Variant**: Variant B (Formula)

### Chapters
Number Systems, Polynomials, Coordinate Geometry, Linear Equations in Two Variables, Introduction to Euclid's Geometry, Lines and Angles, Triangles, Quadrilaterals, Circles, Constructions, Heron's Formula, Surface Areas and Volumes, Statistics, Probability

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Worked examples + practice problems | 60% |
| Concept explanation with intuition | 20% |
| Exam prep (model answers, revision) | 20% |

### Formatting Rules
- **Formulas**: Use code blocks. Example: `Area = √[s(s-a)(s-b)(s-c)]` where `s = (a+b+c)/2`
- **Proofs/Derivations**: Step-by-step with reasoning at each step. Number each step.
- **Geometric figures**: Describe in text ("Draw triangle ABC where AB = 5 cm, angle B = 60°, BC = 4 cm")
- **Worked examples**: Follow this pattern: Given → To Find → Formula → Substitution → Solution → Answer (boxed)
- **Practice problems**: 5 per sub-section, graded easy (2) → medium (2) → hard (1). Answers at end of notes.

### Exam Weightage
| Unit | Marks |
|------|-------|
| Number Systems | 8 |
| Algebra | 17 |
| Coordinate Geometry | 4 |
| Geometry | 28 |
| Mensuration | 13 |
| Statistics & Probability | 10 |

### Common Pain Points
- Abstract proofs feel disconnected from "real math" — always connect to why the proof matters
- Sign conventions in coordinate geometry — use quadrant diagrams described in text
- Confusing surface area with volume — use real-world objects (paint vs. water filling)
- Construction steps seem arbitrary — explain the geometric reasoning behind each step
- Heron's formula feels like a "magic formula" — show why it works intuitively

### Target Word Count
8,000-12,000 words (shorter than narrative subjects because formulas and problems take less prose)

## 2. Science

**NCERT Book**: *Science* (NCERT Class 9)
**Content Type**: Mixed (depends on chapter — see sub-profiles)

### Chapter-to-Branch Mapping

| Chapter | Branch | Structural Variant |
|---------|--------|--------------------|
| Motion | Physics | Variant B (Formula) |
| Force and Laws of Motion | Physics | Variant B (Formula) |
| Gravitation | Physics | Variant B (Formula) |
| Work and Energy | Physics | Variant B (Formula) |
| Sound | Physics | Variant B (Formula) + Variant C (Process) |
| Matter in Our Surroundings | Chemistry | Variant C (Process) |
| Is Matter Around Us Pure | Chemistry | Variant C (Process) |
| Atoms and Molecules | Chemistry | Variant B (Formula) + Variant C (Process) |
| Structure of the Atom | Chemistry | Variant C (Process) |
| The Fundamental Unit of Life | Biology | Variant C (Process/Classification) |
| Tissues | Biology | Variant C (Process/Classification) |
| Diversity in Living Organisms | Biology | Variant C (Process/Classification) |
| Why Do We Fall Ill | Biology | Variant C (Process/Classification) |
| Natural Resources | Biology/Env | Variant C (Process) + Variant A (Narrative) |
| Improvement in Food Resources | Biology | Variant C (Process/Classification) |

### Sub-Profile: Physics Chapters

**Section Emphasis**:
| Component | Weight |
|-----------|--------|
| Concept explanation with real-world examples | 40% |
| Formulas + numerical problems | 30% |
| Exam prep | 20% |
| Experiments/activities | 10% |

**Formatting Rules**:
- Formulas with units in code blocks: `v = u + at` (where v = final velocity in m/s, u = initial velocity in m/s, a = acceleration in m/s², t = time in s)
- Numerical problems: step-by-step with units at every step
- Experiments: Aim → Apparatus → Procedure (numbered) → Observation → Conclusion
- Graphs: Describe axes, shape of curve, what the slope represents

**Common Pain Points**:
- Speed vs velocity vs acceleration — use car/bike examples
- Newton's 3rd law ("if forces are equal and opposite, why does anything move?") — explain the forces act on different objects
- Gravitational force vs weight vs mass — use space/moon examples
- Work = 0 when force is perpendicular — use carrying a bag example

### Sub-Profile: Chemistry Chapters

**Section Emphasis**:
| Component | Weight |
|-----------|--------|
| Concept explanation | 45% |
| Diagrams and classification tables | 25% |
| Exam prep | 20% |
| Activities/experiments | 10% |

**Formatting Rules**:
- Chemical formulas in code: `H₂O`, `CO₂`, `NaCl`
- Atomic structure: Describe electron configurations as text ("2, 8, 1" for sodium)
- Classification: Use tables (Element vs Compound vs Mixture; Metals vs Non-metals)
- Activities: Describe what you'd observe and why

**Common Pain Points**:
- Atom vs molecule vs ion — use analogy (letters → words → charged words)
- Physical vs chemical change — give clear tests (reversibility, new substance formed)
- Mixture vs compound — emphasize fixed ratio vs variable ratio
- Mole concept feels abstract — use "dozen" analogy

### Sub-Profile: Biology Chapters

**Section Emphasis**:
| Component | Weight |
|-----------|--------|
| Classification and process descriptions | 40% |
| Diagrams (text-described) | 30% |
| Exam prep | 20% |
| Real-world applications | 10% |

**Formatting Rules**:
- Classification hierarchies: Use indented lists (Kingdom → Phylum → Class → Order → Family → Genus → Species)
- Cell diagrams: Describe each part and function ("The nucleus is at the center, surrounded by nuclear membrane. It contains chromosomes which carry genetic information.")
- Processes: Numbered steps with "what happens" and "why it happens"
- Disease tables: Disease | Cause | Symptoms | Prevention

**Common Pain Points**:
- Cell organelle functions — too many to memorize without context (relate each to a "factory department")
- Tissue types — confusing plant and animal tissues (use comparison tables)
- Classification hierarchy — seems arbitrary (explain evolutionary basis simply)
- Disease vs infection vs symptom confusion

### Exam Weightage (Science overall)
| Unit | Marks |
|------|-------|
| Matter — Its Nature and Behaviour | 23 |
| Organization in the Living World | 20 |
| Motion, Force and Work | 27 |
| Food & Food Production | 10 |

### Target Word Count
10,000-15,000 words (varies by chapter; Physics numericals and Biology classifications tend to run longer)

## 3. History

**NCERT Book**: *India and the Contemporary World — I*
**Content Type**: Narrative
**Structural Variant**: Variant A (Narrative)

### Chapters
The French Revolution, Socialism in Europe and the Russian Revolution, Nazism and the Rise of Hitler, Forest Society and Colonialism, Pastoralists in the Modern World

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Narrative with cause-effect chains | 40% |
| Timeline + comparison tables | 25% |
| Exam prep (model answers) | 25% |
| Critical thinking + modern relevance | 10% |

### Formatting Rules
- **Timelines**: Chronological table with Year | Event | Significance
- **Cause-Effect**: Use arrow chains: Cause → Event → Immediate Effect → Long-term Impact
- **Comparison tables**: Before vs After, Country A vs Country B, Group X vs Group Y
- **Personality profiles**: Name | Role | Key Actions | Significance (table format)
- **"In Today's World"**: Connect historical events to current issues (e.g., French Revolution → modern democracy movements)

### Common Pain Points
- Dates blur together across chapters — use mnemonics and anchor events
- Ideological differences (liberalism vs socialism vs communism) — use simple spectrum diagrams
- Connecting events to consequences — always state the "so what"
- Source-based questions — teach how to extract information and infer meaning from a passage

### Exam Weightage
History is part of Social Science (80 marks total). History gets approximately 20 marks.

### Target Word Count
12,000-18,000 words (narrative subjects tend to be longer; French Revolution notes v2 was ~16,000 words)

## 4. Geography

**NCERT Book**: *Contemporary India — I*
**Content Type**: Narrative + Visual
**Structural Variant**: Variant A (Narrative) with heavy emphasis on spatial/visual descriptions

### Chapters
India — Size and Location, Physical Features of India, Drainage, Climate, Natural Vegetation and Wildlife, Population

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Descriptive content with spatial context | 35% |
| Map work and visual descriptions | 30% |
| Exam prep | 25% |
| Data/statistics in tables | 10% |

### Formatting Rules
- **Map descriptions**: Describe locations relative to known landmarks ("The Himalayas stretch from the Indus river in the west to the Brahmaputra in the east, forming an arc of about 2,400 km")
- **River systems**: Use flow descriptions (source → tributaries → confluence → mouth)
- **Climate data**: Tables with Region | Temperature Range | Rainfall | Season
- **Vegetation/Wildlife**: Classification tables with Zone | Climate | Flora | Fauna

### Common Pain Points
- Map locations — students struggle to visualize without a physical map (describe relative positions clearly)
- Distinguishing river systems (Himalayan vs Peninsular) — use comparison tables
- Climate factors — too many variables (use cause-effect: latitude → temperature; distance from sea → rainfall)
- Population data interpretation — teach what the numbers mean, not just the numbers

### Exam Weightage
Geography gets approximately 20 marks within Social Science.

### Target Word Count
10,000-14,000 words

## 5. Economics

**NCERT Book**: *Economics* (NCERT Class 9)
**Content Type**: Narrative + Data
**Structural Variant**: Variant A (Narrative) with data emphasis

### Chapters
The Story of Village Palampur, People as Resource, Poverty as a Challenge, Food Security in India

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Concepts with case studies | 35% |
| Data/statistics in tables | 25% |
| Exam prep | 25% |
| Real-world connections | 15% |

### Formatting Rules
- **Case studies**: Summarize Palampur/other examples as: Setting → Situation → Key Economic Concept Illustrated
- **Statistics**: Always present in tables with Year | Indicator | Value | What It Means
- **Economic processes**: Flowcharts in text (Raw materials → Production → Distribution → Consumption)
- **Comparison tables**: Rural vs Urban, Before reform vs After reform

### Common Pain Points
- Economic indicators feel abstract — always tie to "what this means for a family"
- Poverty line calculation — seems arbitrary (explain the calorie basis clearly)
- Factors of production — confusing physical capital with human capital
- "Green Revolution" effects — both positive and negative, students forget one side

### Exam Weightage
Economics gets approximately 20 marks within Social Science.

### Target Word Count
10,000-13,000 words

## 6. Political Science

**NCERT Book**: *Democratic Politics — I*
**Content Type**: Narrative + Conceptual
**Structural Variant**: Variant A (Narrative)

### Chapters
What is Democracy? Why Democracy?, Constitutional Design, Electoral Politics, Working of Institutions, Democratic Rights

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Concept explanation with examples | 40% |
| Case studies and comparisons | 25% |
| Exam prep | 25% |
| Critical thinking (debate-style) | 10% |

### Formatting Rules
- **Definitions**: Always give the formal definition + a simple "in other words" explanation
- **Country comparisons**: Use tables (India vs other democracies, democratic vs non-democratic)
- **Institution functions**: Flowchart-style (Bill introduced → Discussed in Lok Sabha → Passed → Rajya Sabha → President's assent)
- **Rights**: Table format with Right | What It Means | Example | Limitations

### Common Pain Points
- Democracy definitions — students memorize without understanding (use contrasting examples: Zimbabwe, China)
- Institutional roles — confusion between Legislature, Executive, Judiciary
- Electoral process — too many steps (use numbered flow)
- "Rights have limitations" — seems contradictory (explain with clear examples)

### Exam Weightage
Political Science gets approximately 20 marks within Social Science.

### Target Word Count
10,000-13,000 words

## 7. Computer Science

**Content Type**: Applied
**Structural Variant**: Variant E (Applied)

> **Note**: CBSE allows schools flexibility in Computer Science curriculum. Common frameworks include IT (402) and Computer Applications. Adapt based on the source material provided.

### Common Topics
Computer Systems and Organization, Binary/Number Systems, Internet and Web, Introduction to Programming (Python/Scratch), Cyber Safety and Ethics

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Concept explanation with analogies | 40% |
| Practical examples/code | 30% |
| Exam prep | 20% |
| Hands-on exercises | 10% |

### Formatting Rules
- **Code**: Always in code blocks with language specified. Add comments explaining each line.
  ```python
  # Check if a number is even
  num = int(input("Enter a number: "))  # Take input from user
  if num % 2 == 0:                       # Check remainder when divided by 2
      print("Even")                      # Print result if even
  else:
      print("Odd")                       # Print result if odd
  ```
- **Binary/Number systems**: Show conversion steps explicitly (Decimal → keep dividing by 2, write remainders)
- **Input/Output examples**: Always show sample input and expected output
- **Flowcharts**: Text-based using arrows and boxes (Start → Input → Process → Decision → Output → End)

### Common Pain Points
- Binary conversion — which direction to read remainders (use a clear step-by-step with arrows)
- Programming logic — "why does the computer need such specific instructions?" (use recipe analogy)
- Internet protocols — too abstract (use postal system analogy: IP = address, TCP = registered post)
- Cyber safety — students don't see real risk (use age-appropriate case studies)

### Target Word Count
8,000-12,000 words

## 8. English

**NCERT Books**: *Beehive* (main textbook), *Moments* (supplementary reader)
**Content Type**: Language
**Structural Variant**: Variant D (Language)

### Components
Prose chapters (Beehive), Poetry (Beehive), Supplementary prose (Moments), Grammar, Writing Skills

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Comprehension + literary analysis | 30% |
| Grammar rules with examples | 30% |
| Writing format templates | 25% |
| Vocabulary building | 15% |

### Formatting Rules
- **Chapter summaries**: Narrative paragraphs (not just bullet points) — capture the story/argument flow
- **Character analysis**: Table with Character | Traits | Evidence from Text | Role in Theme
- **Poetry**: Stanza-by-stanza explanation, literary devices identified inline, central theme separately
- **Grammar**: Rule → Simple explanation → 3 correct examples → 3 practice sentences with answers
- **Writing formats**: Full templates showing structure:
  - Formal letter: Sender's address → Date → Receiver's address → Subject → Salutation → Body (3 paragraphs) → Complimentary close
  - Diary entry: Date → Salutation (Dear Diary) → Body (what happened + how I felt) → Sign off
  - Story: Title → Opening (setting + characters) → Build-up → Climax → Resolution
- **Vocabulary**: Word | Meaning | Part of Speech | Synonym | Antonym | Use in Sentence

### Common Pain Points
- Unseen passage comprehension — teach scanning vs. close reading strategies
- Tense confusion (present perfect vs past simple) — use timeline visuals described in text
- Formal vs informal letter — students mix up formats (side-by-side comparison)
- Poetry interpretation — "what does the poet mean?" feels subjective (teach: literal → figurative → thematic)
- Reference to context questions — teach: identify speaker → explain context → explain significance

### Exam Structure (English-specific)
| Section | Component | Marks |
|---------|-----------|-------|
| A | Reading Comprehension (2 unseen passages) | 20 |
| B | Writing (letter + story/diary) + Grammar | 20 |
| C | Literature (extract-based + short answer + long answer) | 40 |

### Target Word Count
10,000-14,000 words (per chapter; grammar sections can be shorter)

## 9. Hindi

**NCERT Books**: *क्षितिज (Kshitij)* — काव्य खंड + गद्य खंड, *कृतिका (Kritika)* — supplementary
**Content Type**: Language
**Structural Variant**: Variant D (Language)

### Components
गद्य (Prose) from क्षितिज, काव्य (Poetry) from क्षितिज, Supplementary from कृतिका, व्याकरण (Grammar), लेखन (Writing)

### Section Emphasis
| Component | Weight |
|-----------|--------|
| Literary analysis + comprehension (भावार्थ, सारांश) | 30% |
| Grammar (संधि, समास, अलंकार, पद-परिचय, वाक्य भेद) | 30% |
| Writing formats (पत्र, निबंध, अनुच्छेद) | 25% |
| Vocabulary (शब्दार्थ, मुहावरे, लोकोक्तियाँ) | 15% |

### Formatting Rules
- **Devanagari preservation**: All Hindi terms, verses, and examples in Devanagari script. Transliteration in parentheses only for terms that might be unfamiliar.
- **Poetry (भावार्थ)**: Verse in original → Word meanings (शब्दार्थ) → Explanation (भावार्थ) → Literary devices (अलंकार) → Central theme (केंद्रीय भाव)
- **Prose (सारांश)**: Chapter summary → Character analysis → Key themes → Important quotations with meaning
- **Grammar**: Rule in Hindi → Explanation (can mix Hindi and English for clarity) → 3 examples → 3 practice items
  - संधि: Type (स्वर/व्यंजन/विसर्ग) → Rule → 5 examples with विच्छेद
  - समास: Type → Rule → 5 examples with विग्रह
  - अलंकार: Type (शब्दालंकार/अर्थालंकार) → Definition → 3 examples from poems
- **Writing**: Templates in Hindi:
  - औपचारिक पत्र: प्रेषक का पता → दिनांक → प्राप्तकर्ता → विषय → संबोधन → विषय-वस्तु → धन्यवाद
  - निबंध: भूमिका → मुख्य विषय (3-4 अनुच्छेद) → उपसंहार

### Common Pain Points
- संधि विच्छेद — students apply wrong rules (teach: identify the joining point, check vowel/consonant rules)
- समास identification — confusing types (use comparison table: तत्पुरुष vs कर्मधारय vs द्विगु vs बहुव्रीहि)
- भावार्थ writing — students paraphrase instead of explaining meaning (teach: literal → deeper meaning → poet's intention)
- Formal Hindi letter format — different from English format (provide side-by-side comparison)
- अलंकार identification — students confuse उपमा with रूपक (teach: "like/as" present = उपमा; direct "is" = रूपक)

### Exam Structure (Hindi-specific)
| Section | Component | Marks |
|---------|-----------|-------|
| अ | अपठित बोध (Unseen passages) | 15 |
| ब | व्यावहारिक व्याकरण (Applied grammar) | 15 |
| स | पाठ्यपुस्तक (Textbook — क्षितिज + कृतिका) | 25 |
| द | लेखन (Writing — letter, essay, paragraph) | 25 |

### Target Word Count
10,000-14,000 words (per chapter)
