# Prompt: Generate CBSE Class 9 Study Notes Framework & Process Guide

## OBJECTIVE
Create a comprehensive, reusable framework and step-by-step process guide for transforming any source material into simplified, contextualized study notes optimized for CBSE Class 9 NCERT students (ages 14-15). This framework should enable consistent generation of high-quality educational content that connects concepts to world history, modern relevance, and exam preparation.

---

## CONTEXT & BACKGROUND

### Student Profile
- **Age Group**: 14-15 years old
- **Curriculum**: CBSE Class 9 NCERT
- **Learning Level**: Early secondary education
- **Cognitive Development**: Abstract thinking emerging, need concrete examples
- **Assessment**: Exam-oriented (1-mark, 3-mark, 5-mark answer formats)

### Reference Example
The **French_Revolution_Notes_v2.md** file demonstrates the desired output quality and approach. This document successfully:
- Simplifies complex historical concepts for young learners
- Structures information in multiple formats (narrative, timeline, tables, comparisons)
- Connects topics to world history and modern relevance
- Includes exam-specific answer examples
- Provides quick revision sheets and critical thinking sections

### Goal
Extract the principles and methodology used in French Revolution notes and formalize them into a reusable framework that can be applied to ANY subject (History, Science, Mathematics, Literature, etc.) to generate similarly high-quality study materials.

---

## DELIVERABLES REQUIRED

You must create TWO documents:

### DOCUMENT 1: COMPREHENSIVE FRAMEWORK DOCUMENT
A detailed principles-based guide covering 6 core focus areas:

#### 1. **Pedagogical Principles**
- Learning theory foundations appropriate for 14-15 year olds
- How students at this cognitive level process and retain information
- Balance between conceptual understanding and practical application
- Techniques for connecting abstract concepts to concrete examples and lived experiences
- Memory aids and mnemonics relevant to this age group

#### 2. **Structural & Organizational Principles**
- Mandatory vs. optional sections for study notes
- Optimal information hierarchy and sequencing
- Content organization patterns
- Formatting conventions (headings, bullet points, tables, visual elements)
- Document flow and narrative structure
- How to adapt structure for different subject types

#### 3. **Content Transformation Principles**
- Techniques for simplifying source material without losing accuracy
- Contextualization strategies (connecting to world events, current issues, student experiences)
- How to determine appropriate detail levels
- When to include examples, analogies, and case studies
- Language complexity guidelines appropriate for age 14-15
- Methods for breaking down complex concepts into digestible components

#### 4. **Assessment & Exam Alignment Principles**
- CBSE exam patterns and question types for Class 9
- How to structure content to support different answer formats (3-mark, 5-mark, short answers)
- Keyword and concept identification for exam success
- Techniques for creating revision-efficient materials
- Sample answer construction principles
- Emphasis areas for exam preparation

#### 5. **Accessibility & Learning Styles Principles**
- Language complexity and vocabulary guidelines
- Visual learning support (tables, timelines, diagrams, comparisons)
- Multiple learning pathways (narrative, logical, visual, auditory-friendly)
- Cognitive load management techniques
- Inclusion of diverse perspectives and viewpoints
- Features supporting different learning speeds and styles

#### 6. **Subject Adaptation Principles**
- How framework applies universally across subjects
- Domain-specific variations (History, Science, Mathematics, Literature, Geography, etc.)
- Subject-type adjustments (fact-based vs. concept-based vs. skill-based)
- Content emphasis differences per subject
- Assessment pattern variations by subject

---

### DOCUMENT 2: STEP-BY-STEP PROCESS GUIDE
A practical, sequential workflow for applying the framework to any source material:

Include sections for:
- **Phase 1: Analysis & Planning** - Analyzing source material, identifying key concepts, planning structure
- **Phase 2: Content Extraction** - Selecting information, identifying what's relevant for Class 9 level
- **Phase 3: Simplification & Contextualization** - Rewriting content, adding examples, connecting to modern relevance
- **Phase 4: Structure & Organization** - Organizing into sections, creating hierarchy, formatting
- **Phase 5: Supplementary Materials** - Creating timelines, tables, comparisons, summaries
- **Phase 6: Exam Alignment** - Creating practice answers, revision sheets, question guides
- **Phase 7: Quality Review & Finalization** - Checking age-appropriateness, accuracy, completeness

Each phase should include:
- Specific objectives
- Concrete actions/steps
- Tools or techniques to use
- Quality checks
- Common pitfalls to avoid

1. **Be Comprehensive** - Cover all aspects needed to create high-quality study notes

5. **Be Clear** - Written for someone with no prior context to understand and apply

7. **Be Flexible** - Acknowledge variations needed for different subjects and materials
